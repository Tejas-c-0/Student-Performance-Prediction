# 🎓 Student Performance Prediction System

A Machine Learning web application that predicts whether a student is likely to **Pass** or **Fail** based on academic performance indicators.

Built using **Python, Scikit-Learn, Streamlit, Pandas, NumPy, and Joblib**.

---

## 🚀 Project Overview

Student performance prediction is an important application of Machine Learning in education. This project analyzes student-related factors and predicts academic outcomes.

The system helps identify students who may need additional support and guidance to improve their performance.

---

## 📊 Dataset Information

The dataset contains student-related information such as:

| Feature | Description |
|----------|-------------|
| Hours Studied | Daily study hours |
| Previous Scores | Previous academic scores |
| Pass/Fail | Target Variable |

### 1️⃣ Data Cleaning
- Checked missing values
- Removed duplicates
- Verified data quality

### 2️⃣ Exploratory Data Analysis (EDA)
- Distribution plots
- Pass vs Fail analysis
- Feature relationships
- Correlation analysis

### 3️⃣ Data Preprocessing
- Feature selection
- Train-Test Split
- Data Scaling

### 4️⃣ Model Training

The following Machine Learning algorithms were implemented:

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Student-Performance-Prediction.git
```

Move into the project folder:

```bash
cd Student-Performance-Prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
